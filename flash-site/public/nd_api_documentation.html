Process Inspection Report

Use this API to upload inspection reports. The metadata parameter should contain a stringified JSON object that matches the following schema:

{
  "betterview_pdf_link": "tower.betterview.com/property-profile/13664298/pdf?hash=490AFD16-6119-45D8-881C-48E4C967B0DA",
  "line_of_business": "ho",
  "residence_type": "primary",
  "policy_squate_footage": 2500
}

An example response might be

{
  "policy_number": "D0360513",
  "output_filename": "1744832002_processed/D0360513_INSPECTION_REPORT_5352248_pantheon.pdf",
  "hazards": [
    {
      "name": "Above ground pool without locked fence",
      "explanation": "There is an above ground pool without a fence around it.",
      "severity": 3
    }
  ],
  "is_flagged": true,
  "non_renewal": false,
  "rec_letter": true,
  "itv_flag": false,
  "itv_percentage": 95.3,
  "policy_coverage_a": 425000,
  "inspection_coverage_a": 431000,
  "policy_square_footage": 2500,
  "inspection_square_footage": 2230,
  "total_hazard_score": 3
}
Authorizations:
OAuth2PasswordBearer

required
file
required
string <binary> (File)
metadata
required
string (Metadata)

Responses

https://backend.pantheonai.co/nd/process_inspection_report
